Export / import des données

ok

err

Sauvegarde complète de la base (schéma + données) au format PostgreSQL « custom ». Réservé à l'administrateur ; chaque opération est tracée dans le journal d'audit.

Exporter

Télécharge un fichier .dump restaurable, horodaté. À conserver dans un emplacement sûr (hors serveur).

Importer

Attention : l'import remplace toutes les données actuelles par celles du fichier. Exportez d'abord une sauvegarde. L'opération est atomique (en cas d'échec, la base reste inchangée) et devrait être faite hors période d'activité.

Pour de très grosses bases, préférer les scripts serveur (scripts/sauvegarde-postgres.* / restauration-postgres.sh) — cf. guide d'exploitation.
