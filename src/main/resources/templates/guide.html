Guide d'utilisation — Canevas et import

L'application fonctionne en ligne. En cas de coupure réseau, la saisie se fait hors-ligne dans un canevas Excel, rechargé ensuite.

⬇ Télécharger un canevas vierge

Les étapes

  1. Télécharger le canevas (idéalement le canevas pré-rempli d'une mission créée en ligne).
  2. Remplir le canevas hors-ligne.
  3. Recharger le fichier : l'application le contrôle et affiche un aperçu.
  4. Corriger les anomalies bloquantes, puis recharger.
  5. Le chef de mission vérifie et valide la consolidation.

Règles de saisie

  • Les champs suivis d'un astérisque (*) sont obligatoires.
  • N° d'inventaire : laissez-le vide pour un matériel nouveau (l'app attribue un numéro) ; sinon reportez l'étiquette.
  • Utilisez les menus déroulants (agents, catégorie, type, Oui/Non).
  • Adresses MAC au format AA:BB:CC:DD:EE:FF.
  • Scanners de chèque : numéro de série (pas de MAC).
  • Le relevé réseau se renseigne une seule fois.

Plusieurs agents sur une même mission

Chaque agent remplit son propre fichier : l'application les consolide grâce au N° de mission commun. Répartissez le travail par type de matériel ou par zone pour éviter les doublons.
