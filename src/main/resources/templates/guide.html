Guide d'utilisation - Canevas et import

L'application fonctionne normalement en ligne. En cas de coupure reseau, la saisie se fait hors-ligne dans un canevas Excel, recharge ensuite dans l'application.

Telecharger un canevas vierge

Les etapes

  1. Telecharger le canevas depuis la page d'import.
  2. Remplir le canevas hors-ligne (voir ci-dessous).
  3. Recharger le fichier dans l'application : elle le controle et affiche un rapport.
  4. Corriger les eventuelles anomalies bloquantes, puis recharger.
  5. Le chef de mission verifie, arbitre les doublons et valide la consolidation.

Remplir le canevas

Le canevas contient un onglet « Lisez-moi », un onglet en-tete « 1-Mission et Reseau », un onglet « 2-Membres mission », et un onglet par type de materiel.

  • L'en-tete (N° de mission, poste, dates, chefs) est pre-remplie par l'application. Ajoutez votre matricule de saisisseur et, si besoin, votre zone.
  • Les champs suivis d'un asterisque (*) sont obligatoires.
  • N° d'inventaire : laissez-le vide pour un materiel nouveau (l'application lui attribuera un numero) ; sinon reportez le numero de l'etiquette.
  • Utilisez les menus deroulants (agents, categorie de cable, type, Oui/Non).
  • Adresses MAC au format AA:BB:CC:DD:EE:FF.
  • Scanners de cheque : numero de serie (pas d'adresse MAC).
  • Le releve reseau (etat du cablage, categorie de cable) se renseigne une seule fois.

Plusieurs agents sur une meme mission

Chaque agent peut remplir son propre fichier pour la meme mission : l'application les consolide grace au N° de mission commun. Pour eviter les doublons, repartissez le travail par type de materiel et/ou par zone.

Apres le chargement

L'application produit un rapport : nombre de lignes lues, anomalies bloquantes et avertissements. Les lignes valides peuvent etre consolidees ; les lignes en erreur sont a corriger puis recharger. Le materiel rapproche par numero d'inventaire (ou, a defaut, par MAC / numero de serie) n'est jamais duplique.

A la validation, l'application fige une photo datee de l'inventaire et ressort la liste des nouveaux materiels avec leur numero attribue - a imprimer pour etiqueter les machines.
